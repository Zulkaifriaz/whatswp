Add a floating WhatsApp chat button to your WordPress website that lets visitors message you with one click.

== Description ==

WhatsWP adds a beautifully designed WhatsApp chat button that floats on your website, making it easy for visitors to contact you instantly. Perfect for businesses, customer support, and increasing conversions.

**Features:**
- Fully responsive WhatsApp button
- Choose left or right position
- Three size options (small, medium, large)
- Customizable default message
- Works on all devices
- Lightweight and fast
- Translation ready
- Easy setup

= Why Choose WhatsWP? =
- **Increase Conversions:** Get more leads by making it effortless to contact you
- **Mobile Friendly:** Perfectly sized for all devices
- **Customizable:** Match your brand's style
- **No Coding Required:** Set up in minutes

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/whatswp-click-to-chat` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings > WhatsWP and configure your WhatsApp number and preferences
4. The button will automatically appear on your site

== Frequently Asked Questions ==

= What WhatsApp number format should I use? =
Enter your full phone number with country code but without any symbols. Example: `14151234567` for a US number.

= Can I change the button position? =
Yes! You can choose between bottom left or bottom right in the settings.

= How do I change the button size? =
Go to WhatsWP settings and select from Small (48px), Medium (60px), or Large (72px).

= Can I customize the default message? =
Absolutely! Edit the default message in the plugin settings.

= The button isn't showing up. What should I do? =
1. Check that the plugin is enabled in settings
2. Verify you've entered a valid WhatsApp number
3. Ensure no other plugins are conflicting
